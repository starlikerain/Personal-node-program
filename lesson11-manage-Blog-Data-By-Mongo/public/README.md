## 博客静态资源前端

请配合以下两个项目之一使用

1. [pure-node-notebook](https://github.com/slashhuang/pure-node-notebook)

2. [pure-node-notebook](https://github.com/slashhuang/pure-node-notebook)

## 安装依赖

```bash

	yarn config set registry https://registry.npm.taobao.org
	yarn install

```

## 项目结构

```bash

	|- index 首页
	|- write 写博客
	|- list  博客列表页
	|- about 关于作者
    |- detail 博客详情页面

```
